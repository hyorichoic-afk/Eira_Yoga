# ❄️ Eira Yoga
> **일상 속 균형을 찾는 요가 플랫폼** > **Find Your Quiet (너만의 고요를 찾다)**

---

### 📖 Identity
**Eira**는 ‘눈’을 의미하는 이름으로, 차분히 덮어 정리하고 감각을 맑게 가다듬는 의미를 담고 있습니다.  
일상의 소음을 잠재우고, 온전히 나에게 집중하여 **나만의 리듬**을 되찾는 깊이 있는 요가 경험을 제공합니다.

---

### ✨ Core Values
| Keyword | Concept | Experience |
| :--- | :--- | :--- |
| **❄️ Snow** | **차분한 덮음** | 복잡한 생각과 스트레스를 눈처럼 덮어 내면을 정돈합니다. |
| **💎 Clarity** | **맑은 가다듬음** | 무뎌진 감각을 깨우고 정신을 투명하고 맑게 가다듬습니다. |
| **🧘 Balance** | **일상의 균형** | 흐트러진 리듬을 바로잡아 삶의 완벽한 균형을 완성합니다. |

---

### 🌿 Service Philosophy

* **고요한 정돈 (Quiet Order)**
  - 세상을 덮는 눈처럼, 일상의 번잡함을 가라앉히는 차분한 시퀀스를 지향합니다.
  
* **감각의 회복 (Sensory Recovery)**
  - 맑아진 정신으로 내면의 소리에 귀를 기울이며 본연의 감각을 깨웁니다.

* **지속 가능한 리듬 (Sustainable Rhythm)**
  - 일회성 운동이 아닌, 일상 속에서 꾸준히 나만의 속도를 유지하도록 돕습니다.
